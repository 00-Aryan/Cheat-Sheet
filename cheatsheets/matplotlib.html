01

Figure Setup

Create Figure & Axes daily
python
import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(10, 6))
Multiple Subplots common
python
fig, axes = plt.subplots(nrows=2, ncols=2, figsize=(12, 10))
# Access with axes[0, 0], axes[0, 1], etc.

Plotting

Basic Line Plot daily
python
ax.plot(x_data, y_data, marker='o', linestyle='-', color='blue', label='Data')
Bar Chart common
python
ax.bar(categories, values, color='orange')
03

Axes Customization

Titles and Labels daily
python
ax.set_title("Sales over Time", fontsize=16)
ax.set_xlabel("Date", fontsize=12)
ax.set_ylabel("Revenue ($)", fontsize=12)
ax.legend(loc='upper right')
ax.grid(True, linestyle='--', alpha=0.6)

Saving

Save Figure daily
python
plt.tight_layout()
fig.savefig('plot.png', dpi=300, bbox_inches='tight')
plt.show()
